## コンソールログの確認方法

### ブラウザの開発者ツールを開く
- Windows/Linux: F12 または Ctrl+Shift+I
- Mac: Cmd+Option+I

### コンソールタブを選択
- 左上のタブから「Console」を選択

### ログをフィルタリング
- 検索ボックスに「デバッグ」と入力
- または「===」と入力

### ログをコピー
- ログを右クリック → 「Copy」
- または、ログを選択してCtrl+C（Mac: Cmd+C）

## 期待されるコンソールログ

### 正常な場合（JST）
```
=== 日付クリック（週/日表示）デバッグ ===
clickedDate.getTimezoneOffset(): -540
clickedDate.getHours(): 10
clickedDate.getMinutes(): 0
計算された startTime: 10:00
計算された endTime: 11:00
===================================
```

### 異常な場合（UTC）
```
=== 日付クリック（週/日表示）デバッグ ===
clickedDate.getTimezoneOffset(): 0
clickedDate.getHours(): 1
clickedDate.getMinutes(): 0
計算された startTime: 01:00
計算された endTime: 02:00
===================================
```

この場合、FullCalendarの`timeZone`設定が正しく機能していない可能性があります。
