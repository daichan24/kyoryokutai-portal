# インフォグラフィック用プロンプト置き場

アプリ内の「使い方・ルール説明」を画像で補足するための **画像生成AI向けプロンプト** をまとめています。  
画像はご自身で作成し、フロントの該当画面に差し込んでください。

## 共通ルール（画像仕様）

| 項目 | 推奨 |
|------|------|
| アスペクト比 | **16:9** または **4:3**（横長。本文に合わせてどちらかに統一） |
| 解像度 | 長辺 **1920px** 以上推奨（Retina 表示を考慮） |
| 背景 | 白またはごく薄いグレー。ダークモード用に **反転しやすい配色**（コントラスト十分） |
| 文字 | **日本語**。見出しは短く、本文は **18px 相当以上** の可読サイズを想定 |
| 図形 | **ピクトグラム**・単色アイコン可。線は太め（細線は縮小で消える） |
| 手順 | **番号＋矢印**で左→右または上→下の流れ |
| 表 | 画像内に **枠付き表** でよい（列は3〜5まで） |
| トーン | 自治体・チーム向けの **落ち着いたビジネス調**。装飾過多にしない |

## ファイル一覧（挿入先メモ）

| ファイル | 挿入を想定する画面・位置 |
|----------|-------------------------|
| `01-dashboard.md` | ダッシュボード上部または「カスタマイズ」説明横 |
| `02-schedule-calendar.md` | スケジュール（月表示）の説明エリア |
| `03-mission-project-task.md` | ミッション管理：ミッション・プロジェクト・タスクの関係 |
| `04-event-organizer.md` | イベント主催（旧個人イベント）の協力要請の流れ |
| `05-activity-expense-flow.md` | 活動経費：チェックリスト→役場相談→登録→購入 |
| `06-activity-expense-checklist.md` | 活動経費：〇✕チェックリストの読み方（公式 vs 具体例） |
| `07-sns-weekly-posts.md` | SNS投稿：週ごとのストーリーズ／フィード記録の考え方 |
| `08-wishes-100.md` | やりたいこと100：プロフィールでONにする旨 |
| `09-inspection-fukumei.md` | 復命書：編集・下書き・プレビュー・PDF の流れ |
| `10-nudge-documents.md` | 協力隊細則・周知系（該当する場合） |

各 `.md` に **英語または日本語の画像生成プロンプト全文** と **挿入メモ** を記載してください。
